- ⭐ **2 Stars**
- 🐍 Python-based with modern GUI
- 🎨 Dark-themed interface
- 📥 Supports multiple formats and quality options

A comprehensive desktop application that allows you to search, download, and play music from YouTube.
- ⭐ **2 Stars**
